In this project, a shared map was built for a multi-robot system. This client-server system automatically calculates transformation between different coordinate frames. Building the map includes:

  • Localizing the robot in 3D from a perched camera
  • Particle filter for SLAM and to fuse information
  • A Shared map is built by the server and sent to all the clients

The shared map allows the robots to interact with each other and share their eyes. The video above shows a small application of the dynamic map built by Cozmo. With no prior information about the walls, Cozmo is able to SEE them and cruise through to the target.

Using geometry based computer vision, the positions of Aruco markers were calculated from monocular cameras.
Aruco markers were attached to the robots for pose estimation.
Information from multiple cameras was sent over the network to a server to build a Map of the robot environment.

No prior information about the location of the cameras or the robots is needed.
Odometry readings and camera information were fused using a Rao-Backwellized particle filter for extremely accurate estimates.

This SLAM algorithm increases accuracy the longer the system runs.
