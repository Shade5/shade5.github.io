Perched camera approach for generating a shared world map and simultaneous localization and mapping of a multi-robot system

The video above shows a small application of the dynamic map built by Cozmo. With no prior information about the walls, Cozmo is able to SEE them and cruise through to the target.

Using geometry based computer vision, the positions of Aruco markers were calculated from monocular cameras
Aruco markers were attached the robots for pose estimation
Information from multiple cameras were sent over the network to a server to build a Shared Map of the robot environment. No prior information about the location of the cameras or the robots is needed
Odometry readings and camera information were fused using Rao-Backwellized particle filter for extremely accurate estimates. This SLAM algorithm increases accuracy the longer the system runs
Here the SLAM algorithm was used to locate the walls, only using Cozmo’s camera and movement
Here’s a comparison of the ground truth and the estimated map. Although the Server robot cannot see the walls, it shares its eyes with the client robots and can thus see beyond its normal sight
